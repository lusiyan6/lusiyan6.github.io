java学习笔记

第一章​

文档注释​

  • 格式

    /**
        @author 指定java程序的作者
        @version 指定源文件的版本
        */

    注释内容可被jdk提供的工具javadoc所解析，生成一套以网页文件形式体现的该程序的说明文档

    使用：
    cmd:文件所在地+javadoc+-d+文件命名+-author+-version+本文件名字

    单行注释和多行注释注释的内容不参与编译

预备知识​

  1. java程序编写-编译-运行的过程
  • 编写：我们将编写的java代码保存在以".java"结尾的源文件
  • 编译：使用javac.exe命令编译我们的java源文件。格式：javac+源文件名.java
  • 运行：使用java.exe命令解释运行我们的字节码文件。格式：java+类名
  1. 在一个java源文件中可以声明多个class。但是，只能最多有一个类声明为public，而且要求声明为public的类名必须与源文件名相同
  2. 源文件有几个类，就会生成几个字节码文件
  3. println--先输出后换行
  4. main 固定格式 public static void main(String[]args)
  5. 人机交互方式有图形化界面和命令行方式

java开发工具​

  1. 文本编辑工具 记事本，EditPlus等
  2. 集成开发环境(IDE) Eclipse;myeclipse;intelliJ IDEA;

java应用领域​

java web开发 后台开发

java语言的特点​

  • 面向对象性：
    • 两个要素：类和对象
    • 三个特征：封装、继承、多态
  • 健壮性：
    • 去除c中的指针
    • 自动垃圾回收
  • 跨平台性 一次编译，到处运行 ，可以在不同操作系统跑，归功于java虚拟机

第二章​

关键字和保留字​

  1. 关键字都是小写的

  2. goto,const是保留字

  3. 标识符：可以自己命名的字符。包名、类名、变量名等。

    命名规则:
    24个英文字母大小写，0-9，_或$中选
    不可数字开头
    不可与关键字保留字相同
    区分大小写
    不能包含空格

    命名规范：
    包名：xxxyyyzzz 多单词组成时所有字母都小写
    类名，接口名：XxxYyyZzz  多单词组成时首字母大写
    变量名、方法名：xxxYyyZzz  多单词组成时,首字母小写，第二个单词开始每个单词首字母大写
    常量名：XXX_YYY所有字母大写，多单词用＿相连

变量的使用​

  1. 变量必须先声明赋值后才能使用
  2. 变量都定义在作用域内
  3. 使用变量名访问这块区域的数据
  4. 变量类型

  • 按数据类型来分：

    • 基本数据类型

      • 数值类型：整型:byte\short\long\int(注意他们的范围)，浮点型:float\double声明long变量，必须以"l""L"结尾 ,声明float变量，必须以F,f结尾

      • 字符型char(两个字节)

        1. 声明一个字符
        2. 转义字符
        3. 直接使用Unicode值来表示字符常量

      • 布尔型boolean

    • 引用数值类型 初始化值都是null

      • 类（字符串）
      • 数据接口
      • 数组

  • 按声明的位置分:

    • 所有变量：分为成员变量和局部变量。
    • 成员变量：实例变量(不以static修饰)和类变量(以static修饰)
    • 局部变量：方法局部变量(在方法内定义)和代码块局部变量(在代码块内定义)

基本数据类型之间的运算规则​

前提：这里讨论只是7种基本数据类型变量之间的运算。不包括boolean类型

  1. 自动类型提升
范围大与范围小的运算，运算结果会自动转换成范围大的类型，可以用更大范围数据类型接收结果
byte、short、char->int->long->float->double
char参与运算会转化成ASIIC码,强制类型转换成int型也会转化成ASSCII码。 
当byte char short三种类型做运算，结果都是int
  1. 强制类型转换 需要使用强转符(),可能导致精度损失。

String类型变量的使用​

  • String 属于引用数据类型，字符串。String可以为空，但char类型不可为空

  • String可以和8种基本数据类型做运算，只能做连接运算，运算结果是String类型

  • java中String不可强制类型转换

  • 字符串与int之间的相互转化

    • 将String转化成int

      1. int i=Integer.parseInt(String);
      2. int i=Integer.valueOf(String).intValue();

    • 整型转化成String

      1. String s=String.valueOf(int);
      2. String s=Integer.toString(int);
      3. String s=""+i;

运算符注意事项​

  1. %运算，结果与前面的数符号相同
  2. ++，--，+=，-=，不会改变本身变量的数据类型
  3. 比较运算符
  • instanceof检查是否是类的对象，生成布尔类型的结果。

  1. 逻辑运算符，操作布尔类型的变量

    &逻辑与

    &&短路与

    1. 自己和自己异或为零
    2. 0和任何数异或为任何数
    3. 满足交换率和结合率
位运算符(了解)
    操作的都是整型的数据 (二进制数) 
    <<左移    左移1位乘以2 空缺位补0   
    >>右移    最高位是0补0，最高位是1补1   除以2  
    >>>无符号右移  都补0
    &与 
    |或 
    ^异或 
    ~取反
  1. 三元运算符
  • (条件表达式)?表达式1：表达式2;

第三章​

数组​

  • 数组属于引用数据类型的变量。
  • 数组的元素，既可以是基本数据类型，也可以是引用数据类型。

一维数组​

  • 声明：int[]ids;
  • 初始化(只有这两种写法)
    ids=new int[]{1001,1002,1003,1004};//静态
    ids=new int[5];//动态
  • 一旦完成初始化数组长度确定，长度无法修改

数组的分类​

1.按照维数 2.按照数组元素的类型：基本数据类型元素的数组、引用数据类型元素的数组

获取数组的长度​

//属性length
System.out.println(ids.length);

数组元素的默认初始化值​

  1. 基本数据类型

数组元素是整型：0 数组元素是浮点型：0.0 数组元素是char型：ASCII码是零的符号 数组元素是布尔类型：false

  1. 引用数据类型

    数组元素是String型：NULL 数组的内存解析 栈和堆 栈存放局部变量(数组的地址) 堆存放数组

二维数组​

行元素是数组(引用数据类型) 声明 int [][]arr; 初始化 arr=new int[][]{{1,2,3},{44,55,66}}; arr=new int[3][2];//一定要写行，列可写可不写 arr1[2]=new int[10]; 初始化值的问题 针对初始化方式1：比如int[][]arr=new int[4][3]; 外层元素的初始化值是地址值 内层元素的初始化值是与一维数组相同 针对初始化方式2：比如int[][]arr=new int [4][]; 外层元素的初始化值是NULL 内层元素的初始化值不能调用，调用会报错，空指针异常

数组中涉及的常见算法

  1. 数组元素的赋值(杨辉三角、回形数等)
  2. 求数值型数组中元素的最大值、最小值、平均数、总和等
  3. 数组的复制、反转、查找(线性查找、二分法查找)
  4. 数组元素的排序算法 二分法 前提：所要查找的数组必须有序 冒泡和快排

各种内部排序方法性能比较 1.从平均时间而言：快速排序最佳。但在最坏情况下时间性能不如堆排序和归 并排序。 2.从算法简单性看：由于直接选择排序、直接插入排序和冒泡排序的算法比较 简单，将其认为是简单算法。对于Shell排序、堆排序、快速排序和归并排序 算法，其算法比较复杂，认为是复杂排序。 3.从稳定性看：直接插入排序、冒泡排序和归并排序时稳定的；而直接选择排 序、快速排序、 Shell排序和堆排序是不稳定排序 4.从待排序的记录数n的大小看，n较小时，宜采用简单排序；而n较大时宜采 用改进排序。 3.4 数组中涉及到的常见算法：排序算法排序算法的选择 (1)若n较小(如n≤50)，可采用直接插入或直接选择排序。 当记录规模较小时，直接插入排序较好；否则因为直接选择移动的记录数少于直 接插入，应选直接选择排序为宜。 (2)若文件初始状态基本有序(指正序)，则应选用直接插入、冒泡或随机的快速排 序为宜； (3)若n较大，则应采用时间复杂度为O(nlgn)的排序方法：快速排序、堆排序或 归并排序。

数组异常 数组脚标越界异常(ArrayIndexOutOfBoundsException) 空指针异常(NullPointerException) int []arr=null; arr[0];

数组工具类的使用 1 boolean equals(int[] a,int[] b) 判断两个数组是否相等。 2 String toString(int[] a) 输出数组信息。 3 void fill(int[] a,int val) 将指定值填充到数组之中。 4 void sort(int[] a) 对数组进行排序。 5 int binarySearch(int[] a,int key) 对排序后的数组进行二分法检索指定的值。 如果返回值为负则没有找到
